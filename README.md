# Thermal Printer

A driver class for the CSN-A2-T thermal receipt printer from Adafruit connected to an imp via the serial port.

## Class Usage

Constructor: ThermalPrinter(*uart, baud*)

## Class Methods

reset()

print(*printString*)

load(buffer)

pull()

feed(*lines*)

setJustify(*value*)

writeMode()

setBold([*value*])

setUnderline([*value*])

setReverse([*value*])

setUpdown([*value*])

setEmphasized([*value*])

setDoubleHeight([*value*])

setDoubleWidth([*value*])

setDeleteLine([*value*])
